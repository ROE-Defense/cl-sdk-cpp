# Contributing to cl-sdk-cpp

Thank you for your interest in contributing to `cl-sdk-cpp`!

## Guidelines
- All contributions must maintain the mathematically accurate 59-channel HD-MEA mapping.
- The C-core must continue to bypass the Python GIL and use zero-overhead structures.
- Follow the existing C99 and C++17 coding styles.
- Submit Pull Requests against the `main` branch.
- Ensure the GitHub Actions CI pipeline passes before requesting a review.
